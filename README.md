### Developing

#### Prerequisites
You'll need Node.js installed, as well as Git and optionally Docker

#### Setup Commands

```
git clone git@github.com:Lissy93/stack-match.git
cd stack-match
npm i
npm run dev
```

---

### Deploying

#### Option 1
Follow the developing instructions above. Then run `npm run build` to compile output<br>
You can then upload the contents of `./dist` to any web server, CDN or static host.

#### Option 2
Fork the repo, and import into Vercel, Netlify or any static hosting provider of your choice.

#### Option 3
Build the Docker image from the Dockerfile with `docker build -t stack-match .`<br>
And then start the container, by running `docker run -p 8080:80 stack-match`
